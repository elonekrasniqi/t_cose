Detailed Description

A structure to hold a pool of struct t_cose_parameter. See T_COSE_PARAM_STORAGE_INIT(). Typically this structure and an array ot t_cose_parameter are allocated in the decode context so the return parameters from a decode have a lifetime of the decode context. They can also be allocated on the stack or elsewhere as long a s the lifetitime is as needed.
