Detailed Description

This is primarily for backwards compatibility with t_cose v1. Instead use the linked list of parameters returned and functions like t_cose_find_parameter_alg_id().

This holds the common header parameters defined in section 3 of RFC

  1. It was the only way that parameters were returned in t_cose 1.x which did not support any parameters but these. For t_cose 2.x parameters are returned as linked lists of struct t_cose_parameter.

Approximate size on a 64-bit machine is 80 bytes and on a 32-bit machine is 40.
